## 🔧 PHASE 1: Domain & Hosting Setup

### 1.1 Update Your Domain

**Status:** ⚠️ TODO

Currently set to: `HustleTeacher.vercel.app`

**To use a custom domain (e.g., hustleteacher.com):**

1. Register domain at GoDaddy, Namecheap, or similar
2. In Vercel dashboard: Settings → Domains → Add → Enter your domain
3. Update DNS records to point to Vercel
4. Update `astro.config.mjs`:
   ```
   site: 'https://hustleteacher.com'
   ```

### 1.2 Set Up Vercel Deployment

**Status:** ✅ CONFIGURED

Your `vercel.json` is already set up with:
- Security headers (CSP, X-Frame-Options, HSTS)
- Cache control for assets
- Environment variables defined

**Deploy:**
```bash
npm run build
# Push to git and Vercel will deploy automatically
```

### 1.3 Enable HTTPS (SSL)

**Status:** ✅ AUTOMATIC

Vercel automatically provides free SSL certificates on all domains.

## 📊 PHASE 2: Analytics & Tracking Setup

### 2.1 Google Analytics 4

**Status:** ⚠️ TODO

1. Go to https://analytics.google.com
2. Create a new property called "Hustle Teacher"
3. Get your Measurement ID (format: `G-XXXXXXXXXX`)
4. Add to `.env`:
   ```
   PUBLIC_GA_ID=G-XXXXXXXXXX
   ```
5. Redeploy your site
6. Test: Go to Real-time → Active users should show up

### 2.2 Google Search Console

**Status:** ⚠️ TODO

1. Go to https://search.google.com/search-console
2. Add property: Select "URL prefix" → Enter `https://HustleTeacher.vercel.app`
3. Verify via:
   - HTML tag (copy the verification code)
   - Add to `public/` folder OR
   - DNS record
4. Submit sitemap: `/sitemap-index.xml`
5. Monitor crawl errors and index status

### 2.3 Microsoft Clarity (Heatmaps)

**Status:** ⚠️ TODO

1. Go to https://clarity.microsoft.com
2. Sign in with Microsoft account
3. Create new project: "Hustle Teacher"
4. Copy Project ID and add to `.env`:
   ```
   PUBLIC_CLARITY_ID=your-project-id
   ```
5. Redeploy your site
6. Check dashboard for visitor heatmaps

### 2.4 Bing Webmaster Tools

**Status:** ⚠️ TODO

1. Go to https://www.bing.com/webmaster

## 📧 PHASE 3: Email Marketing

### 3.1 Set Up Mailchimp

**Status:** ⚠️ TODO

1. Go to https://mailchimp.com (create free account)
2. Create audience named "Hustle Teacher Subscribers"
3. Go to Settings → API Keys → Copy your API key & Server (e.g., `us1`)
4. Create an audience and get Audience ID
5. Add to `.env`:
   ```
   PUBLIC_MAILCHIMP_AUDIENCE_ID=your-audience-id
   PUBLIC_MAILCHIMP_SERVER=us1
   ```
6. Set up confirmation email: Campaigns → Automations → Opt-in workflow
7. Test: Use the newsletter form on your site
8. Monitor: Mailchimp dashboard shows all subscribers

**Note:** NewsletterSignup component is ready to use across your site.

**Status:** ⚠️ TODO

2. Create site:
